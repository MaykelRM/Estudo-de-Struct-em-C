# 📚 Sistema de Gerenciamento de Alunos em C

Este projeto é um sistema simples de gerenciamento de alunos desenvolvido em linguagem C. A aplicação funciona via terminal e permite:

- ✅ Cadastrar alunos com nome, ID e três notas  
- ✅ Calcular a média individual automaticamente  
- ✅ Listar todos os alunos cadastrados  
- ✅ Buscar aluno pelo ID  
- ✅ Calcular a média geral da turma  

O sistema utiliza **struct**, vetores, manipulação de strings (`strcpy`, `strcat`) e organização modular com funções para cada funcionalidade do menu.

## 🎯 Objetivo do Projeto

Projeto desenvolvido com foco na prática de:

- Estruturas (`struct`)  
- Vetores  
- Laços de repetição  
- Funções  
- Manipulação de strings  
- Lógica de programação em C
