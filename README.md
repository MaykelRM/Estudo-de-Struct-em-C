# Estudo-de-Struct-em-C
Sistema simples de gerenciamento de alunos desenvolvido em C, executado via terminal. Permite cadastrar alunos com ID e três notas, calcular média individual, listar todos os alunos, buscar por ID e calcular a média geral da turma. Utiliza struct, vetores, funções e manipulação de strings para organização dos dados.
